# US 3: Extraction de fichiers

Commande exécutée :

```bash
sha256sum Manifest.xml
```

Résultat obtenu :

```BASH
fe4a0005356c4f1bb09538fb55e5a38d26e05f87790298695e44897de91b00c7  Manifest.xml
```

sha256sum classes.dex
